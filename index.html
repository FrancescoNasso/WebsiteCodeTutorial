You can use the <p> tag in order to write paragraphs. Paragraphs have a default styling which you can further customize using CSS.

Bolding

You can make things bold with the <b> tag. Alternatively, you can use the <strong> tag to denote exactly the same thing.

Italics

You can make your content italicized using <i>. The same effect can be achieved using <i> which stands for emphasis.

Subscripts

You can use HTML to denote superscript (e.g., 12) using the <sup> tag. You can also use subscript (e.g., H20) by using the <sub> tag.

Links and Page Anchors (⚓)

Absolute URLs

Links can be used to call out to link to external websistes or to sections within a page of a site. Links start with an opening link tag <a> and ends in a closing link tag </a>.

Example: Google

Source Snippet

<a href="www.google.com">Google</a>

Relative URLs

Relative URLs are used for linking to parts of the same site.

Source Snippet

<a href="index.html">Google</a>

Emails

A mail to can be used to get the user to mail.

Source Snippet

<a href="mailto:dante5541@aol.com">Email Me!!!</a>

Target

Targets can be used to specify how things open. Target="_blank" is used to tell a browser to open a page in a new tab.

Example: Google (opens in a new window)

Source Snippet

<a href="www.google.com" target="_blank">Google (opens in a new window)</a>

Images

Image tags

Image tags are used to display images on a website. An image tag <img> usually has a few accomping tags to make the content of the image known if the reader is using a screen reader. <alt>

Source Snippet

<img src="http://cdn2-www.dogtime.com/assets/uploads/gallery/havanese-dog-breed-pictures/6-sidestanding.jpg" title="cute dog" alt="This is a picture of a cute dog">

Fun Fact: It is considered best practice to use alt and title for all image files. This helps to make your website more accessible to users.

Height and Width

You can control the size of an image on a browser by specifying its height and width as atributes. Here is the same picture minimized to 200px height and 200px width.

Source Snippet

<img src="http://cdn2-www.dogtime.com/assets/uploads/gallery/havanese-dog-breed-pictures/6-sidestanding.jpg" title="cute dog" alt="This is a picture of a cute dog" height="200" width="200">

Fun fact: CSS is more frequently used to control the height and width of images.

Image Figures and Captions

A recent development it HTML5 is the use of figures and captions to link together images with their captions. Figures are are designated with the figure tag <figure> and captions are designated with the figure caption tag <figcaption>.

Source Snippet

<Figure>
      <img src="https://i.pinimg.com/originals/3a/53/90/3a5390b170efc769a9b6f434480b1501.jpg" title="space cow" alt="a picture of a cow floating in space">
      <figcaption> This is a picture of a cow floating in space.</figcaption>
</Figure>

Tables

HTML can be used to build the skeleton for tables. Tables are created using the table tag <table>. Rows are specified using <tr> and boxes within the rows are represented using the table data tag <td>.

Example:

Table Data 1-1 Table Data 1-2 Table Data 1-3
Table Data 2-1 Table Data 2-2 Table Data 2-3
Table Data 3-1 Table Data 3-2 Table Data 3-3
Table Headers

A table header <th> is used to specify that a data point within the header. Using this gives the creater greater control when integrating CSS properties.
