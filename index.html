Example: Google

Images

Image tags

Image tags are used to display images on a website. An image tag <img> usually has a few accomping tags to make the content of the image known if the reader is using a screen reader. <alt>

Source Snippet

<img src="http://cdn2-www.dogtime.com/assets/uploads/gallery/havanese-dog-breed-pictures/6-sidestanding.jpg" title="cute dog" alt="This is a picture of a cute dog">

Fun Fact: It is considered best practice to use alt and title for all image files. This helps to make your website more accessible to users.

Height and Width

You can control the size of an image on a browser by specifying its height and width as atributes. Here is the same picture minimized to 200px height and 200px width.

Source Snippet

<img src="http://cdn2-www.dogtime.com/assets/uploads/gallery/havanese-dog-breed-pictures/6-sidestanding.jpg" title="cute dog" alt="This is a picture of a cute dog" height="200" width="200">

Fun fact: CSS is more frequently used to control the height and width of images.

One way of styling elements of a table are to use table tags <thead>, <tbody>, <tfoot>. <thead> should go around the headings of a table. <tbody> should go around all elements of the table excluding the headers and footers. <tfoot> should go on the bottom of a table. For larger tables, this is important as you can use <tfoot> and <thead>to make sure content is always visible.

Forms

HTML can be used to create forms such as textboxs, dropdown menus, etc. However, you will need the backend of your website to be able to handle responses for these forms to work.

Forms are designated with the <forms> tag. Additionally, for each form you should specify the method you want to transmit data with. <get> is used for small transfers of information, whereas post is used for larger pieces of information (like textbox data), as well as information that is private.

For many forms, <input> is used to specify the type of form you want displayed to the user. See the textbox example below.
