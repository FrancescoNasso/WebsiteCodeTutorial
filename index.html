HTML5

You can use the <p> tag in order to write paragraphs. Paragraphs have a default styling which you can further customize using CSS.

You can use heading tag <h1> - <h6> to designate headings. Browsers attribute their own size and font to each heading.

You can make things bold with the <b> tag. Alternatively, you can use the <strong> tag to denote exactly the same thing.

You can make your content italicized using <i>. The same effect can be achieved using <i> which stands for emphasis.

You can use HTML to denote superscript (e.g., 12) using the <sup> tag. You can also use subscript (e.g., H20) by using the <sub> tag.

Line breaks <br /> are really good for making things appear on new lines.

Blockquotes <blockquotes> are used to specify longer quotes. When combined with CSS you can use it to apply a specific stylization to your larger quotes.

"It was the best of times it was the worst of times." - The opening to A Tale of Two Cities.
You can also specify smaller quotes using the quote attribute <quote>. For smaller quotes use this! <quote> - Francesco Nasso

Definition <dfn> tags can be used to specify a definition.

coyling: sickeningly cute.

You can specify addresses by using the address tag <address>:

example-email@example.org

One of the most useful features of HTML is the ability to specify lists <li>. There are three main types of lists: Ordered lists <ol>, Unordered lists <ul>, and Definition lists <dl>.

  1. One for the money
  2. Two for the show
  3. Three to get ready
  4. Four to go

A definitional list is very special. You use definitional terms to specify the term and the definition tags to specify the definition.

Sushi
A type of Japanese cuisine typified by raw fish.
Sashimi
Sliced raw fish that is served with condiments such as shredded dicon radish or ginger root, wasabi and soy sauce.
